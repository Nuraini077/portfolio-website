Website ini merupakan sebuah personal portfolio berbasis web yang dikembangkan menggunakan HTML dan CSS. Tujuan dari pembuatan website ini adalah untuk menampilkan profil pribadi, keterampilan, serta hasil karya secara digital.

Website ini terdiri dari beberapa bagian utama, yaitu:

-About Me: berisi informasi latar belakang dan profil singkat
-Skills: menampilkan kemampuan yang dimiliki
-Portfolio: berisi kumpulan karya yang pernah dibuat
-Contact: menyediakan informasi kontak

Website dirancang dengan tampilan yang sederhana, responsif, dan user-friendly sehingga mudah diakses oleh pengguna dari berbagai perangkat.
